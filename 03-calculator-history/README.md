# Java Swing Calculator

03-calculator with a classic 4×4 button layout and an additional bottom **History** button.  
Every calculation is saved to `history.txt`, and you can view all previous calculations through the History menu.

## Features
- Classic 4×4 calculator layout
- Buttons: digits 0–9, +, -, *, /, C, =
- Bottom bar with **History** button
- Saves each calculation automatically to `history.txt`
- Displays previous calculations in a popup window
- Prevents multiple decimals in one number
- Handles divide-by-zero errors
