# Java Swing Calculator

02-calculator built using Java Swing.  
It supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- GUI interface
- Buttons for digits 0–9
- Operators: +, -, *, /
- Clear (C) and equals (=) buttons
- Input display

## Project Structure
/src  
 ├── Main.java  
 └── Calculator.java  
README.md

## How to Run
1. Install Java JDK 8 or higher.
2. Compile:
   javac Main.java Calculator.java
3. Run:
   java Main

## Requirements
- Java JDK 8 or newer

## Notes
- Division by zero will display Infinity.
